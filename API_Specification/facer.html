This call retrieves the list of all users

get /users

This call retrieves the list of all users

Secured by basic
  • Response
  • Security

HTTP status code 200

This is the successful response to the get users call.

Body

Media type: application/json

Type: array of User Details

Items: User Details

  • id: (string - minLength: 1 - maxLength: 10)
  • firstName: required (string)

    This is the user's first name

  • lastName: required (string)

    This is the user's last name

  • role: required (string)

    This is the role of the user

  • preferences: (object)

    This is the frequency with which messages are pulled from the user device

    • messagePullFrequency: required (number)

      This is the frequency in seconds with which messages are pulled from the user device

    • quietHours: required (object)

      This object represents the quiet hours of the user

      • startTime: required (time-only)

        This is the start time of the quiet hours.

      • endTime: required (time-only)

        This is the end time of the quiet hours.

Example:

[
{
  "id": "1000",
  "firstName": "Christopher",
  "lastName": "Vanderford",
  "role": "Project Manager",
  "preferences": {
    "messagePullFrequency" : 10,
    "quietHours":{
      "startTime": "21:00:00",
      "endTime": "06:00:00"
    }
  }
},
{
  "id": "1001",
  "firstName": "Joseph",
  "lastName": "Wroblewski",
  "role": "Architect",
  "preferences": {
    "messagePullFrequency" : 10,
    "quietHours":{
      "startTime": "22:30:00",
      "endTime": "06:00:00"
    }
  }
}]
